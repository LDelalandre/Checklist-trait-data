# Generate Appendices of *Delalandre et al. 2026*

**Article**: *From FAIR to MEGaFAIR: A checklist to future-proof trait data in ecology and evolution*

**Authors**: Léo Delalandre, Jules Segrestin, Karim Barkaoui, Cyrille Violle, and Eric Garnier

`data` contains the detailed checklist in csv format and an html template.

`scripts` contains R scripts used to generate the Appendix 1 (the detailed checklist formatted in PDF format) and Appendix 2 (the operational checklist, with each element preceded by a tickbox, in HTML format).


All scripts run in R studio. See details in `sessionInfo.txt`.
